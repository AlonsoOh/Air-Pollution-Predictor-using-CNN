- 프로젝트명 : 머신러닝을 활용한 내일의 미세먼지수치 예측 서비스 
  
- 팀명 : KDK 
  
- 결과물 브랜드명 : 엄마눈이따가와 
  
  
# 풀고자 하는문제(Pain Point)
  
  과거 봄에만 만연했던 황사가 이제는 계절을 타지않고 시도때도없이 서울 하늘을 뒤덮는 시대가 왔습니다.   
  다들 한번쯤은 외출하려고 마음먹었는데 밖에 창문너머 노란하늘을 본 순간 결심이 흔들린적이 있을 것입니다.   
  
  미세먼지는 인간의 호흡기, 심혈관 뿐만아니라 여러가지 암의 원인이 되는것은 밝혀진 사실입니다.   
  저희는 이런 미세먼지의 심각성에대해 미리 경각심을가지고 대비할수있도록 사람들에게 구체적인 내일의 미세먼지 수치를 알려주는   
  '미세먼지 알리미'를 통해 편의를 제공하고자 합니다.   
  
  근본적으로 대기 질의 개선이라는 해결책이 불가능한 지금시점에서 우리는 최대한으로 대비하고 피해를 입지않도록 예방해야합니다.    
  이것을 돕기위해 '미세먼지 알리미'는 내일의 미세먼지 예측 수치를 사용자에게 알려주고,   
  사용자가 야외활동을 자제하고 마스크를 착용하는 등 실질적인 방어양식을 갖출수있도록 합니다.   


# 어떻게 풀것인가?
  
- 아두이노 미세먼지 키트[~~여기 만들기 가이드~~](http://eduiot.kr/wp-content/uploads/2017/11/%EC%95%84%EB%91%90%EC%9D%B4%EB%85%B8-%EB%AF%B8%EC%84%B8%EB%A8%BC%EC%A7%80%EC%B8%A1%EC%A0%95%EA%B8%B0-3D-KIT-%EB%A7%8C%EB%93%A4%EA%B8%B0-v2.pdf)를 이용해 비교적 정확한 대기상태를 측정하기위해 고도가 높은 아파트 창문밖에 설치 (컴터랑 연결)
  
- 여기에 같이 실시간 대기정보를 알수있는 풍속센서, 습도센서, 온도센서도 설치
  
-  이 센서들에서는 매 시간, 매 분 단위로 미세먼지, 온도, 풍속 , 습도를 측정, 실시간 대기정보 데이터 받아옴
  
-  이 수치들은 AWS ec2서버에 있는 데이터베이스로 쌓임(with TIMESTAMP)
  
-  데이터가 제법 쌓이면 PYTHON KERAS를 이용해 Convolution Neural Network 만듬
  
-  CNN에 시간(24시기준), 온도, 풍속, 습도 등의 대기정보를 입력하고 결과값으로 미세먼지 수치 나오도록 train
  
-  내일 일기예보에서 시간별로 온도, 습도, 풍속등 가져와서 모델에 입력-> 시간별 미세먼지수치 예측값 나옴
  
-  (이부분은 내일 일기예보에 의존할건지, 아니면 CNN을 좀더 고급지게 트레이닝해서 입력값 없이도 내일 수치 나오게할수있을지??)
  
-  이 CNN 프로그램은 서버에 상주, 소비자가 쓸 수 있는(분석결과를 알려주는) <안드로이드 앱 or 웹 페이지> 까지 따로 제작
-  +추가로 WHO에서 정한 미세먼지 등급체계에 따라 위험레벨 안내
